HTML

HTML: hypertext markup language 超文本标记语言

HTML骨架

DTD

DTD(Document Type Definition)

meta标签代表配置

字符集


标签分类

  • 单表签/双标签

  • 容器级/文本级

文本的空白折叠现象

在普通文字之间，如果有空格、换行、缩进导致的空白，在浏览器中加载时会被折叠成一个空格显示，这就是空白折叠现象

空格：可以使用字符实体替换书写，在代码中书写&nbsp;替换空格

HTML常见标签

h系列标签

六级标题，有h1,h2,h3,h4,h5,h6. 都是双标签，容器级标签

p标签

英文:paragraph

双标签，文本级标签

作用：添加一个完整段落的语义

img标签

<img src="../1.jpg" alt="">

英文: image

单标签, 文本级标签

作用: 在指定位置插入一张图片

img标签的属性

  1. src: 作用是引入图片路径

  2. alt: 图片加载不出来时候的替换文本

锚点标签 a

英文 anchor

双标签，文本级标签

作用: 在指定位置添加一个超级链接，从而实现相应的跳转

a标签几个属性

  1. href: hypertext reference

<a href="http://www.baidu.com">跳转到百度</a>

  1. target: 作用是是否在新标签打开链接，值一定是”_blank”

<a href="12_a标签的跳转.html" target="_blank">跳转到12文件</a>

  1. 页面锚点的使用

     锚点的作用: 实现点击超级链接从实现页面内的跳转
 方法一
 <a href="#jbxx">基本信息</a> // 跳转到锚点
 <a name="jbxx"></a> //不含信息的隐藏锚点
 方法二
 <a href="#jbxx">基本信息</a>
 <h3 id="jbxx">基本信息</h3>

无序列表

ul+ li

英文: ul(ulordered list) li(list item)

用法

双标签，容量级标签，需要注意的是，ul内部嵌套且只能嵌套li，li内部可以嵌套任何标签

有序列表

ol + li

英文: ol(ordered list) li(list item)

双标签，容量级标签，ol内部必须只能嵌套li

定义列表

dl + dt + dd

英文 dl(definition list) dt(definition term) dd(definition description)

双标签，容量级标签，dl内部只能存放dt和dd，dt和dd同级

dt用作定义主题，dd对dt进行描述解释，多个dd对应在其之上的一个dt

表格

table + tr + td

英文: table tr(table rows) td(table dock)

table/tr定义行/td定义单元格

表格分区

表单元素

form

容器级，内部存放可输入的控件，如果输入的表单需要提交到数据，所有控件需要被from表单包裹

method: 方法，指提交数据的方法，属性值为post 和 get

action: 是数据提交的位置

input标签

被form包裹

通过type扩展

输入框

value: 设为默认显示的内容

placeholder: 如果没有value时提示用户的文字占位符

<input type="text" placeholder="请输入用户名">

密码框

<input type="password" placeholder="请输入密码">

单选框

复选框

文本域

textarea

rows: 定义文本域的可视区域有几行，单位是数字

cols: 当前行显示的字节数量(以英文为准), 单位是数字

下拉菜单

select + option

div和span

div和span是常用的布局标签，俗称盒子

div: 分割跨度大跨度 容器级标签，双标签，布局分割

span: 小区域小跨度 用来进行文字分割
